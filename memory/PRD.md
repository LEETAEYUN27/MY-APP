# 정보비서 (Info-Secretary) PRD

## 원본 문제 정의
맞춤형 정보 업데이트 앱. 타깃: 정보취약계층(노인 등), 맞춤형 정보를 원하는 사람들.
작동 원리: 고객 니즈 입력 → 정보 소스 설정 → AI 분류 → 인앱 알림 송부

## 아키텍처
- **Backend**: FastAPI + MongoDB + emergentintegrations (OpenAI GPT-5.2)
- **Frontend**: React + Tailwind CSS + Shadcn UI
- **Auth**: JWT (httpOnly cookies) + bcrypt
- **Data**: 웹 스크래핑 (Naver News, Google News RSS)
- **AI**: GPT-5.2로 뉴스 분류 및 요약

## 사용자 페르소나
1. 정보취약계층 (노인) - 큰 글씨, 심플한 UI 필요
2. 바쁜 직장인 - 맞춤형 정보를 빠르게 확인하고 싶은 사람
3. 투자자 - 기업 소식 및 공시 정보를 추적하고 싶은 사람

## 핵심 요구사항
- JWT 기반 로그인/회원가입
- 관심사 키워드 CRUD (청년혜택, 연예인, 기업소식, 상품, 기타)
- 정보 소스 설정 (Naver News, Google News, DART)
- 웹 스크래핑 + AI(GPT-5.2) 기반 정보 수집 및 분류
- 인앱 알림 시스템
- 한글 UI 전체 구현
